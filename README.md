# Friends of Hotels

Immersive hotel booking management site for the Friends of hotel discount program.

## Pages
- Home: `index.html`
- Hotels: `hotels.html`
- Offers: `offers.html`
- Admin mockup: `admin/index.html`

## Deploy to Cloudflare Pages
1. Connect this repository to Cloudflare Pages.
2. Set build command to none.
3. Set output directory to `/`.
4. Deploy the `main` branch.

## Notes
This is a static mockup built with HTML, CSS, and lightweight JavaScript for quick client presentation and iteration.
